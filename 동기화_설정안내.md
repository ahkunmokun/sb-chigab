# ☁️ 기기 간 동기화 설정 (한 번만)

폰·PC에서 같은 데이터를 보려면, Supabase에서 **딱 2가지**만 설정하면 됩니다.
사용하는 프로젝트: `hbirnbnoqaoqizlrphoi` (schedule-app과 같은 Supabase 계정)

---

## 1단계 — 데이터 보관 테이블 만들기 (SQL 1회 실행)

1. https://supabase.com/dashboard 로그인
2. 프로젝트 **hbirnbnoqaoqizlrphoi** 선택
3. 왼쪽 메뉴 **SQL Editor** → **New query**
4. 아래를 통째로 붙여넣고 **Run** (▶) 클릭

```sql
-- 가계부 동기화용 테이블 (사용자별 1행, 전체 데이터를 jsonb로 보관)
create table if not exists public.ledger_state (
  user_id    uuid primary key references auth.users(id) on delete cascade,
  state      jsonb not null,
  updated_at timestamptz not null default now()
);

-- 행 보안: 본인 데이터만 읽기/쓰기 (공개 키를 써도 남의 데이터는 못 봄)
alter table public.ledger_state enable row level security;

create policy "own select" on public.ledger_state
  for select using (auth.uid() = user_id);
create policy "own insert" on public.ledger_state
  for insert with check (auth.uid() = user_id);
create policy "own update" on public.ledger_state
  for update using (auth.uid() = user_id) with check (auth.uid() = user_id);
```

"Success. No rows returned" 이 뜨면 성공입니다.

---

## 2단계 — 이메일 인증 끄기 (가입을 즉시 되게)

1. 왼쪽 메뉴 **Authentication** → **Sign In / Providers** (또는 Settings)
2. **Email** 항목에서 **"Confirm email"** 옵션을 **OFF**로
3. Save

> 이걸 끄면 앱에서 이메일·비밀번호를 넣자마자 바로 가입·로그인됩니다.
> (끄지 않으면 가입 후 이메일의 확인 링크를 눌러야 로그인됩니다.)

---

## 3단계 — 앱에서 로그인

1. https://ahkunmokun.github.io/sb-chigab/ 열기 (폰·PC 각각)
2. **설정 → ☁️ 기기 간 동기화**
3. **이메일 + 비밀번호(6자 이상)** 입력 → **로그인/가입**
4. 폰과 PC에서 **같은 이메일·비밀번호**로 로그인하면 끝!

이제 한쪽에서 거래를 추가하면 자동으로 올라가고, 다른 기기에서 앱을 열거나 **"🔄 지금 동기화"** 를 누르면 내려받습니다.

---

## 동기화 동작 방식 (참고)
- 데이터 변경 시 자동으로 클라우드에 업로드 (1.5초 후)
- 앱을 열면 클라우드에서 최신본을 내려받음
- **최신 수정본이 이김** (rev 시각 기준). 두 기기에서 동시에(오프라인으로) 고치면 나중 것이 이기고 한쪽은 덮어써질 수 있으니, 한 기기씩 쓰는 걸 권장.
- 인터넷이 없으면 그냥 그 기기에서만 저장되고, 연결되면 다시 동기화됩니다.
- 공개 앱에 들어간 키는 **publishable(공개용)** 키라 안전합니다. 실제 데이터 보호는 위 RLS(행 보안)가 합니다.
